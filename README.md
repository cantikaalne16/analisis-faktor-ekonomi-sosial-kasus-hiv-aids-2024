# Analisis Faktor Ekonomi Sosial Kasus HIV & AIDS 2024
## Project Akhir Data Wrangling Semester Gasal 2025/2026 S1 Sains Data Universitas Negeri Surabaya

### Deskripsi Project:
Project ini melakukan proses data wrangling terhadap beberapa dataset (PDF dan CSV) yang berkaitan dengan kasus HIV/AIDS serta faktor sosial ekonomi di Indonesia. Proses meliputi scraping PDF, data cleaning, integrasi dataset, dan eksplorasi sederhana.

### Fitur Utama
- Scraping data HIV/AIDS dari file PDF menggunakan PyMuPDF.
- Ekstrak tabel PDF dengan pdfplumber & Tabula-py
- Menggabungkan dataset sosial-ekonomi dari berbagai sumber:
    1. Data kasus HIV & AIDS di Indonesia tahun 2024 >> https://kemkes.go.id/id/profil-kesehatan-indonesia-2024
    2. Data jumlah penduduk miskin di Indonesia tahun 2024 >> https://www.bps.go.id/id/pressrelease/2025/01/15/2401/persentase-penduduk-miskin-september-2024-turun-menjadi-8-57-persen-.html
    3. Data indeks tingkat pengangguran terbuka di Indonesia tahun 2024 >> https://opendata.jabarprov.go.id/id/dataset/tingkat-pengangguran-terbuka-berdasarkan-provinsi--di-indonesia 
    4. Data rata-rata lama sekolah di Indonesia tahun 2024 >> https://www.bps.go.id/id/statistics-table/2/NDE1IzI=/-metode-baru--rata-rata-lama-sekolah--tahun-.html
- Data cleaning dan standarisasi kolom.
- Pembuatan variabel Rasio HIV/AIDS.
- Eksplorasi data (EDA) dan visualisasi.
